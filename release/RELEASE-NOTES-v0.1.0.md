# PinBase v0.1.0

## 版本定位

首个可交付 MVP 版本。

## 已包含能力

- Popup 网站列表
- 搜索与分类筛选
- 添加当前网站
- 编辑、删除、撤销
- 分类新增、重命名、删除
- 快捷键收录：`Alt + Shift + K`
- 页面内快捷键 toast
- 已收藏状态 badge
- Options 页面基础状态查看

## 构建与发布物

- 扩展产物目录：`dist/`
- 发布压缩包：`release/PinBase-v0.1.0.zip`

## 发布前建议

- 在真实浏览器中再完成一轮完整手工回归
- 确认 popup、快捷键、分类管理、toast 和 options 页面都正常
