## 2. 当前项目目标

交付一个可在 Chromium 浏览器中运行的 PinBase 插件 MVP，支持：

- 收藏当前网站
- 搜索网站
- 分类筛选
- 网站卡片展示
- 编辑、删除与撤销
- 快捷键收录
- Badge 状态提示
- 页面内 toast 提示

## 8. 统一验收方式

### 命令验收

1. 运行 `npm run check`
2. 运行 `npm run test`
3. 运行 `npm run build`
4. 确认三条命令全部通过

### 浏览器验收

1. 打开 `chrome://extensions`
2. 对 PinBase 点击“重新加载”
3. 打开任意普通网页，点击插件图标，确认 popup 正常打开
4. 点击“添加当前网站”，确认可以保存，并出现绿色成功 toast
5. 点击任意卡片，确认能打开网站
6. 删除一个卡片，确认出现金色 toast，并可点击“撤销”恢复
7. 在左侧分类上右键，确认可看到“重命名 / 删除分类”
8. 对某个分类点“重命名”，确认分类名、卡片分类、弹窗分类选项都会同步更新
9. 对某个分类点“删除分类”，确认先出现二次确认弹窗；确认后该分类消失，相关网站自动归到“其他”
10. 按 `Alt + Shift + K`，确认页面右下角出现快捷键收录 toast
11. 再按一次同样快捷键，确认出现“已收录过”的普通提示
12. 打开 options 页面，确认能看到统计信息、快捷键说明和分类管理区
