交付一个可在 Chromium 浏览器中运行的 PinBase 扩展 MVP，支持：

- 收藏当前网站

- 分类筛选
- 网站卡片展示
- 编辑、删除与撤销
- 快捷键收录
- Badge 状态提示
- 页面内 toast 提示

## 7. 统一验收方式

### 命令验收

1. 运行 `npm run check`
2. 运行 `npm run test`
3. 运行 `npm run build`
4. 确认三条命令全部通过

### 浏览器验收

1. 打开 `chrome://extensions`
2. 对 PinBase 点击“重新加载”
3. 打开任意普通网页，点击插件图标，确认 popup 正常打开
4. 点击“添加当前网站”，确认可保存并出现成功提示
5. 点击卡片，确认能打开网站
6. 删除卡片后点击“撤销”，确认可恢复
7. 在左侧分类上右键，确认可重命名和删除
8. 删除分类时，确认会先出现二次确认
9. 按 `Alt + Shift + K`，确认页面内出现快捷键收录提示
10. 打开 options 页面，确认能看到统计信息和分类管理区域
